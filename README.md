# 🤖 Бот-тренер по дисциплине

Telegram-бот на Python для отслеживания привычек: вода, зарядка, холодный душ.
Утренние напоминания, вечерние отчёты, статистика и «умный тренер» на DeepSeek.

## Возможности
- ⏰ **7:00** — «Доброе утро!» + план дня (время настраивается)
- ✅ Кнопки «Сделал(а)» для каждой привычки
- 🌙 **00:00** — «Время отчёта о дне» + анализ
- 📊 Статистика: серия 🔥, % за неделю, текстовый график
- 🧠 Умный тренер (DeepSeek): хвалит/критикует, даёт советы, отвечает как живой человек
  - Когда ты выполнил все привычки → похвала
  - Пропустил → мотивация / критика
  - Пишешь боту → общение как с тренером
- ⚙️ Настройки времени подъёма и отчёта
- 💾 Встроенная SQLite (`habits.db`)

## 🚀 Локальный запуск (для теста)

```bash
cd discipline-bot
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

pip install -r requirements.txt
python bot.py
```

Ключи уже прописаны в `.env`. Открой бота в Telegram и нажми **/start**.

## 🟣 Деплой на Railway (проще всего)

1. Залей папку `discipline-bot` в репозиторий на GitHub.
   > ⚠️ Файл `.env` НЕ заливай (он уже в `.gitignore`).
2. Зайди на https://railway.app → **New Project → Deploy from GitHub repo**.
3. Variables → добавь переменные (значения возьми из `.env`):
   `BOT_TOKEN`, `DEEPSEEK_API_KEY`, `DEEPSEEK_BASE_URL`, `DEEPSEEK_MODEL`, `TZ`.
4. Railway сам поставит зависимости и запустит `python bot.py` (см. `Procfile`).
5. Готово. Бот работает, пока хватает бесплатного кредита.

## 🟢 Деплой на Render (Free)

1. Залей репозиторий на GitHub (без `.env`).
2. https://render.com → **New → Web Service** → выбери репозиторий.
3. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
   - **Instance Type:** Free
4. Environment → добавь переменные из `.env`.
5. **Чтобы не засыпал:** скопируй URL сервиса (вида `https://xxx.onrender.com`)
   и добавь его в бесплатный пингер https://uptimerobot.com (HTTP-проверка раз в 5 мин).
   Встроенный health-check (`/`) для этого уже есть в коде.

## 🏆 Деплой на Oracle Cloud (реально бесплатно навсегда)

Самый надёжный «вечный» бесплатный вариант — отдельная виртуалка.

1. Зарегистрируйся на https://www.oracle.com/cloud/free/ (нужна карта для верификации, деньги не снимут).
2. Создай **Always Free** инстанс: *Compute → Instances → Create*,
   образ **Ubuntu**, форма **Always Free eligible** (VM.Standard.A1.Flex или E2.1.Micro).
3. Подключись по SSH и выполни:
   ```bash
   sudo apt update && sudo apt install -y python3-pip python3-venv git
   git clone <твой-репозиторий> && cd discipline-bot
   python3 -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   nano .env        # вставь свои ключи
   ```
4. Запусти как сервис, чтобы работал вечно (автоперезапуск):
   ```bash
   sudo tee /etc/systemd/system/habitbot.service > /dev/null <<'EOF'
   [Unit]
   Description=Discipline Habit Bot
   After=network.target

   [Service]
   WorkingDirectory=/home/ubuntu/discipline-bot
   ExecStart=/home/ubuntu/discipline-bot/.venv/bin/python bot.py
   Restart=always
   User=ubuntu

   [Install]
   WantedBy=multi-user.target
   EOF

   sudo systemctl daemon-reload
   sudo systemctl enable --now habitbot
   sudo systemctl status habitbot
   ```
5. Всё. Бот работает 24/7 и сам перезапускается после перезагрузки.

## 🤖 Google Gemini: Умный тренер

Бот использует **Google Gemini API** как умного тренера. Это лучше, чем DeepSeek:
- ✅ **Бесплатный уровень** есть (пробный период)
- ✅ **Дешевле** — ~$0.1–0.3/месяц на обычное использование
- ✅ **Быстрее** — ответы приходят мгновенно
- ✅ **Лучшее качество** на русском языке

**Что умеет умный тренер:**
- Анализирует твои привычки
- Хвалит за выполнение, критикует за пропуски
- Общается как реальный человек (не робот)
- Даёт советы по дисциплине

### Как это работает

Бот становится «живым тренером»:
1. Выполнил все привычки → **живая похвала**
2. Пропустил → **мотивация и советы**
3. Пишешь боту что-то → **отвечает как человек**
4. Вечерний отчёт → **анализирует день**

Если Gemini не работает, бот всё равно отвечает заглушками — всё функционально работает.

### Проверка, работает ли Gemini

```
/test_api
```

Бот попробует запрос и скажет:
- ✅ **Работает** — видишь ответ
- ❌ **Не работает** — смотри логи (на Railway: вкладка Logs)

### Если Gemini не работает

Возможные причины:
1. **Ключ неправильный** — проверь `.env` (GOOGLE_API_KEY)
2. **API не включен** — нужно включить в Google Cloud Console
3. **Модель не доступна** — проверь, что `gemini-2.5-flash` включена

### Как дебагить

На **Railway** смотри логи:
1. Откройся в Railway → выбери сервис
2. Вкладка **Logs**
3. Ищи:
   ```
   → Запрос к Gemini: Я выполнил все привычки...
   ← Gemini OK: Молодец, так держать!
   ```

### Если совсем не хочется платить

Бот работает **полностью без Gemini**:
- Кнопки работают ✅
- Напоминания приходят ⏰
- Статистика считается 📊
- Fallback-ответы на вопросы 💬

Главное — привычки отслеживаются!
